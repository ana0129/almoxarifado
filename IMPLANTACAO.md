# Guia de Importação e Infraestrutura

## Pré-requisitos de ambiente (Local)
para rodar esse projeto no seu computador voê precisa ter instalado:
[ ] PHP (versão 8.2 ousuperior);
[ ] Composer (Gerenciador de pacotes do PHP)
[ ] Filament (Framework para o laravel)
[ ] Noje.js & NPM
[ ] Banco de dados MySQL

## CheckList de Instalação
[ ] Criar a estrutura da base de Dados no Laravel
[ ] Instalar o Filament
[ ] Configurar o arquivos de variáveis de ambiente ('.env).
[ ] Rodar as Aplicaçõea do banco de dados ('migrations').
